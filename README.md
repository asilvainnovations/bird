# BIRD 2026–2035 Document Generation System

**Bangsamoro Investment Roadmap Development (BIRD)** - Production Document Generator

## 🎯 Purpose

Generate professional, multi-format documents (DOCX, Google Docs, PDF) for the **Bangsamoro Investment Roadmap Development (BIRD) 2026–2035**, aligned with MTIT-BARMM branding and standards.

## 📋 Requirements

- **Node.js**: v18+
- **npm**: v9+
- **Operating System**: Linux/macOS/Windows (WSL)
- **Memory**: 2GB+ RAM for PDF rendering

## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/barmm-mtit/bird-document-generator.git
cd bird-document-generator

# Run automated setup
chmod +x setup.sh
./setup.sh

# Edit .env with credentials
nano .env

# Generate all formats
npm run deploy
